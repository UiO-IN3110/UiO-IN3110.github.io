Quantifiers
===========

-----------  ----------------------
Quantifyier  Description
-----------  ----------------------
?            0 or 1
*            0 or more
+            1 or more
{n}          exactly 'n'
{n,}         'n' or more
{,n}         'n' or less
{n,m}        'n', 'm' or in between
-----------  ----------------------
